🧺 Achetons Burkinabè
Plateforme web interactive de promotion et de mise en relation pour les producteurs et artisans locaux du Burkina Faso. Projet universitaire réalisé dans le cadre d'un travail de groupe (5 personnes).
🚀 Fonctionnalités principales
Page d'accueil : Présentation du concept, mise en valeur des filières locales (karité, sésame, miel, coton, artisanat).
Catalogue dynamique : Affichage d'une liste de produits locaux avec détails, prix et vendeurs.
Recherche et filtres instantanés : Filtrage par catégorie et recherche textuelle en temps réel par nom ou par vendeur.
Galerie Lightbox : Agrandissement visuel des produits avec navigation interactive.
Formulaire de contact vendeur : Permet d'envoyer un message ciblé à un producteur avec validation dynamique des champs.
Mode sombre (Dark Mode) : Confort visuel optimisé avec sauvegarde des préférences via le localStorage.
🛠️ Technologies utilisées
HTML5 : Structure sémantique des pages.
CSS3 : Styles personnalisés, variables CSS (:root), design responsive et gestion du mode sombre.
JavaScript (Vanilla) : Logique applicative, gestion des états, manipulation du DOM et filtrage dynamique.
👥 Répartition des tâches (Groupe de 5)
Personne 1 : Structure de base et page d'accueil.
Personne 2 : Design global, palette de couleurs et mode sombre.
Personne 3 : Intégration des données produits et galerie lightbox.
Personne 4 : Système de recherche instantanée et filtres par catégories.
Personne 5 : Formulaire de contact sécurisé et intégration finale.
